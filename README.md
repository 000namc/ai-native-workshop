# AI-Native 워크숍

> AI-Native 업무 환경을 이해하고, 코딩 에이전트를 세팅하여 활용해본다.

**대상:** 비개발 직군 (PM, 디자이너)
**시간:** 총 2시간

---

## 하위 목표

1. AI 툴을 활용할 수 있는 방법들을 이해한다
2. 개발 환경의 기본을 이해한다
3. 코딩 에이전트의 기본 사용법을 이해한다

---
