# 102. Chrome Extension — QA 자동화

> Session 1: AI 활용

## 이것은 무엇인가요?

Chrome Extension(크롬 확장 프로그램)을 활용해 반복적인 QA 작업을 자동화하는 방법입니다.

## 왜 채팅이 아니라 Extension인가요?

- **Claude 채팅**: Claude 서버에서 동작 → 회사 내부망에 접근할 수 없음
- **Chrome Extension**: 내 브라우저에서 동작 → 내부망 서비스에 직접 접근 가능

내부 서비스를 QA하려면 Extension이 필요합니다.
