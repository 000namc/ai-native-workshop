# 301. 맥락 관리 — agent.md 파일

> Session 3: 코딩 에이전트 기본

## 이것은 무엇인가요?

CLAUDE.md(또는 agent.md)는 코딩 에이전트에게 프로젝트의 맥락을 알려주는 파일입니다.
사람에게 업무 인수인계서를 주는 것과 같습니다.

## 왜 중요한가요?

- 에이전트는 매번 새로운 대화를 시작합니다
- 맥락 파일이 없으면 매번 처음부터 설명해야 합니다
- 잘 작성된 맥락 파일 = 일 잘하는 에이전트

## 좋은 맥락 파일 작성법

### 포함할 내용

```markdown
# 프로젝트 이름

## 기술 스택
- 어떤 언어/프레임워크를 사용하는지

## 프로젝트 구조
- 폴더가 어떻게 구성되어 있는지

## 규칙
- 코드 스타일, 네이밍 컨벤션 등

## 현재 상태
- 지금 어디까지 진행되었는지
```

### 좋은 예시 vs 나쁜 예시

| 나쁜 예시 | 좋은 예시 |
|-----------|-----------|
| "웹앱 만들어줘" | "Node.js + Express로 REST API 서버를 만들어줘. SQLite를 DB로 사용해." |
| (맥락 없음) | "src/ 아래에 코드를 작성하고, build/ 아래에 Docker 설정을 넣어줘." |
