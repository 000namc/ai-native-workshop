# 105. Webhook & Cron — 자동화 트리거

## Webhook — 이메일 초안 작성하기

### 이것은 무엇인가요?

Webhook은 특정 이벤트가 발생했을 때 자동으로 작업을 실행하는 트리거입니다.

### 왜 유용한가요?

- 특정 조건이 충족되면 자동으로 이메일 초안을 작성할 수 있습니다

## Cron — 매주·매일 업무 요약 정리하기

### 이것은 무엇인가요?

Cron은 정해진 시간에 반복적으로 작업을 실행하는 스케줄러입니다.

### 왜 유용한가요?

- 매일 아침 어제의 업무 요약을 자동으로 받을 수 있습니다
- 매주 금요일 주간 보고 초안을 자동 생성할 수 있습니다
