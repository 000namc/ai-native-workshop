# 000. 워크숍 소개

## 이 워크숍은 무엇인가요?

AI-Native 업무 환경을 이해하고, 코딩 에이전트를 세팅하여 직접 활용해보는 워크숍입니다.

이 워크숍은 **Claude**를 중심으로 진행합니다. 강연자가 개인적으로 추천하는 도구이며, AI 에이전트를 활용하는 방식은 이 외에도 다양합니다.

> 다른 선택지 예시: Antigravity, Cursor, Lovable, Gemini Code Assist, Bolt ...

## 누구를 위한 워크숍인가요?

- PM, 디자이너 등 비개발 직군
- 코딩 경험이 없어도 괜찮습니다

## 워크숍이 끝나면 무엇을 할 수 있나요?

1. AI 도구를 업무에 활용하는 다양한 방법을 알게 됩니다
2. 개발 환경(터미널, Git, Docker)의 기본 개념을 이해합니다
3. 코딩 에이전트에게 원하는 결과물을 만들어달라고 요청할 수 있습니다

## 워크숍이 끝나면 공감할 수 있는 것들

1. **Claude Code가 '로컬 환경'에서 동작한다는 것** — 내 파일, 내부망에 직접 접근 가능하기 때문에 다른 도구보다 자유도가 크다
2. **AI에게 "잘 시키는 법"이 있다는 것** — 맥락(CLAUDE.md), 구체적 요구사항, 작업 단위 분리
3. **에이전트가 하는 일을 읽을 수 있다는 것** — 터미널 명령어, git 로그를 보고 "지금 뭘 하고 있는지" 이해
4. **반복 업무를 AI에게 맡길 수 있다는 것** — 한 번 세팅하면 Webhook/Cron으로 자동화
