# 00. 워크숍 소개

## 이 워크숍은 무엇인가요?

AI-Native 업무 환경을 이해하고, 코딩 에이전트를 세팅하여 직접 활용해보는 워크숍입니다.

## 누구를 위한 워크숍인가요?

- PM, 디자이너 등 비개발 직군
- 코딩 경험이 없어도 괜찮습니다

## 워크숍이 끝나면 무엇을 할 수 있나요?

1. AI 도구를 업무에 활용하는 다양한 방법을 알게 됩니다
2. 개발 환경(터미널, Git, Docker)의 기본 개념을 이해합니다
3. 코딩 에이전트에게 원하는 결과물을 만들어달라고 요청할 수 있습니다

## 진행 순서

| 세션 | 주제 | 시간 |
|------|------|------|
| Session 1 | AI 활용 | 30분 |
| Session 2 | 개발 환경 기본 | 1시간 |
| Session 3 | 코딩 에이전트 기본 | 30분 |
