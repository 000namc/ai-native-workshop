# Session 2 — 개발 환경 기본 (1시간)

개발 환경의 핵심 개념을 이해합니다.

## 왜 이걸 배우나요?

AI가 코드를 대신 짜주는 시대지만, 그 코드를 **실행하고, 관리하고, 배포하는 환경**은 여전히 사람이 이해해야 합니다.

이번 세션에서 다루는 CLI, Git, Docker는 개발을 시작하기 위한 **필수 지식 3가지**입니다:

- **CLI** — 개발 도구는 대부분 터미널에서 동작합니다. 클릭이 아닌 명령어로 컴퓨터를 다루는 법을 알아야 합니다.
- **Git** — 코드는 계속 바뀝니다. 변경 이력을 추적하고, 팀원과 안전하게 협업하려면 버전관리가 필수입니다.
- **Docker** — "내 컴퓨터에서는 되는데?"를 없애줍니다. 누구나 같은 환경에서 코드를 실행할 수 있게 해줍니다.

코딩 에이전트를 활용하더라도 이 세 가지를 모르면 에이전트가 만든 결과물을 다룰 수 없습니다.

## 이런 걸 배웁니다

- 터미널(CLI)로 컴퓨터를 조작하는 방법
- Git으로 코드 버전을 관리하는 방법
- Docker로 개발 환경을 세팅하는 방법
