# 101. Claude 도구 이해하기

> Session 1: AI 활용

## 왜 먼저 알아야 하나요?

Claude에는 여러 도구가 있고, 각각 할 수 있는 일이 다릅니다.
이걸 먼저 이해해야 이후 세션에서 "왜 이 도구를 쓰는지"가 이해됩니다.

## Chat / Cowork / Code

| 도구 | 동작 위치 | 방식 |
|------|-----------|------|
| **Chat** | Claude 서버 | 물어보면 답한다 (Q&A) |
| **Cowork** | Claude 서버 (로컬 파일 편집 가능) | 일을 시키면 계획 세우고 복잡한 작업을 수행한다 (에이전트) |
| **Code** | 로컬 환경 | Cowork와 같은 에이전트인데, **내 컴퓨터에서** 파일/터미널을 직접 다룬다 |

## 핵심 포인트

- **Chat**: 빠른 질문/답변에 적합
- **Cowork**: 복잡한 작업을 맡길 때 — 계획을 세우고 단계적으로 수행
- **Code**: 내 컴퓨터에서 직접 코드 작성, 파일 편집, 명령어 실행
